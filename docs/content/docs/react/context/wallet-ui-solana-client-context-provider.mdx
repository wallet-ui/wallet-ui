---
title: WalletUiSolanaClientContextProvider
description: The provider for the WalletUiSolanaClientContext.
---

The `WalletUiSolanaClientContextProvider` is a React context provider that creates and provides a `SolanaClient` instance.

This component is used internally by the main `WalletUi` provider and you probably won't need to use it directly.

### Props

| Name           | Type                       | Description                               |
| -------------- | -------------------------- | ----------------------------------------- |
| `children`     | `React.ReactNode`          | The content to render.                    |
| `urlOrMoniker` | `SolanaClientUrlOrMoniker` | The URL or moniker of the Solana cluster. |

### Example

This example shows how you could use the provider.

```tsx
import { WalletUiSolanaClientContextProvider } from '@wallet-ui/react';

function MyApp({ children }) {
    return <WalletUiSolanaClientContextProvider urlOrMoniker="devnet">{children}</WalletUiSolanaClientContextProvider>;
}
```
