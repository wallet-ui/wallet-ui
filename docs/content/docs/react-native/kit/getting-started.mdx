## Overview

`@wallet-ui/react-native-kit` is the React Native SDK for Wallet UI built on top of `@solana/kit`. It provides components and hooks for connecting your mobile dApp with wallets that support the Mobile Wallet Adapter standard.

If you are starting a new project, this is the recommended SDK to use.
